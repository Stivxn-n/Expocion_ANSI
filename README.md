# Exposición ANSI

Página web educativa sobre el estándar ANSI aplicado a la programación.

## ¿De qué trata?

Explica qué son los códigos de escape ANSI, cómo funcionan los colores
en la terminal, su historia y en qué lenguajes de programación se usan.

## Contenido

- ¿Qué es ANSI?
- Historia desde 1918 hasta hoy
- Tabla de colores con sus códigos
- Ejemplos en Python, Bash y JavaScript
- Usos en el mundo real

## Tecnologías

- HTML5
- CSS3
